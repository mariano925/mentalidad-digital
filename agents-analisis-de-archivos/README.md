# Multimedia Design Thinking 🎬☁️

Este repo es un ejercicio práctico para aprender y divertirme con **HTML, CSS y JavaScript**, integrando multimedia (audio, video, imágenes) y un pequeño **quiz interactivo**.  
No es un producto final, sino un espacio de pruebas y aprendizaje.

## Contenido
- Secciones de audio y video sobre ciberseguridad y la nube.
- Un quiz simple para reforzar lo aprendido.
- Estilos con efecto *Glassmorphism* para darle un look moderno.
- 🎉 Confeti de celebración al responder todo correctamente.

## Objetivo
Practicar cómo combinar multimedia, interactividad y diseño en un sitio web estático.  
El repo funciona como mi cuaderno de pruebas: voy corrigiendo rutas, agregando bloques y ajustando estilos.

## Estado
🚧 En construcción. Es un prototipo de práctica, no un proyecto terminado.
